# 通用审查模块(所有场景适用)

本模块包含所有音乐版权服务合同共通的审查内容,不区分场景。
场景专项审查完成后,必须再叠加本模块进行通用审查。

## 一、合同主体审查

### 1.1 主体资格

- ✅ 双方主体名称与营业执照/统一社会信用代码是否一致
- ✅ 签约代表是否有授权(法定代表人或授权委托书)
- ✅ 甲乙方经营范围是否覆盖合同业务
- ✅ 是否为关联交易(若是,需特别披露)

### 1.2 主体信息完整性

必须包含:
- 公司全称
- 统一社会信用代码
- 法定代表人/负责人
- 注册地址 / 通讯地址(若不一致需特别注明)
- 联系电话、电子邮箱
- 开户银行及账号(涉及付款时)

## 二、合同基本结构审查

### 2.1 合同名称

- 合同名称是否准确反映合作实质?
- 避免"挂羊头卖狗肉"(如名为"服务合同"实为"授权合同")

### 2.2 鉴于条款 / 前言

- 是否清晰描述双方合作背景?
- 鉴于条款是否会产生意外的法律拘束力?

### 2.3 定义条款

- 关键术语是否定义?(如"音乐作品"、"使用"、"传播"、"关联公司"等)
- 定义是否清晰、无歧义?

## 三、通用商务条款审查

### 3.1 付款条款


| 审查项 | 要点 |
|-------|------|
| 付款金额 | 大小写一致、是否含税 |
| 付款币种 | 人民币/外币 |
| 付款方式 | 银行转账/其他 |
| 付款节点 | 节点清晰、可执行 |
| 发票 | 类型(增专/普票)、税率、开具时间 |
| 延迟付款 | 违约金标准 |


### 3.2 税费承担

- 各自税费各自承担?
- 是否有代扣代缴义务?
- 是否涉及跨境支付(需考虑预提所得税)?

### 3.3 通知与送达

- 通知方式:书面/电子邮件/EMS?
- 送达地址是否明确?
- 送达视为送达的拟制规则?

## 四、合同履行通用条款

### 4.1 履行期限

- 履行起止时间是否明确?
- 阶段性履行节点是否清晰?
- 时间表达是否无歧义("X日"是工作日还是自然日?)

### 4.2 变更与解除

- 变更需经双方书面同意?
- 单方解除的情形是否明确、合理?
- 协商解除的程序?
- 解除后的后果处理?

### 4.3 不可抗力

- 不可抗力定义是否合理?
- 是否包含"政府行为"、"政策变化"?(对乙方有利)
- 通知义务及证明文件要求?
- 不可抗力持续时间过长时的解除权?

## 五、保密条款审查

### 5.1 保密信息范围

- 商业秘密的定义是否清晰?
- 是否包含未来产生的信息?
- 是否有"标记为保密"的要求?

### 5.2 保密义务

- 保密期限(合同期内 + 合同结束后 X 年)?
- 接触保密信息的人员范围?
- 例外情形(公开信息、独立开发、法律要求披露等)?

### 5.3 违反保密的后果

- 违约金标准?
- 实际损失赔偿?
- 是否包含禁令救济?

## 六、知识产权通用条款

### 6.1 权属确认

- 各自先有知识产权归属?
- 合作中产生的新知识产权归属?
- 第三方知识产权使用许可链?

### 6.2 侵权处理

- 第三方主张侵权时的处理流程?
- 抗辩责任分担?
- 赔偿责任分担?

## 七、违约责任通用审查

### 7.1 违约金条款

- 违约金标准是否合理?(过低无威慑,过高可能被法院调整)
- 是否区分一般违约、根本违约?
- 违约金与实际损失的关系?

### 7.2 损害赔偿

- 直接损失 vs 间接损失?
- 赔偿范围是否包括律师费、诉讼费、保全费、公证费等?
- 是否设置赔偿上限?

### 7.3 救济措施

- 继续履行?
- 补救措施?
- 合同解除?
- 多种救济措施可否累加适用?

## 八、争议解决条款审查

### 8.1 适用法律

- 中国法律(中华人民共和国大陆地区法律)?

### 8.2 管辖


| 方式 | 优劣 |
|------|------|
| 仲裁(机构名称、规则) | 保密性好,一裁终局,但费用高 |
| 法院诉讼(原告/被告/合同履行地等) | 选择需对乙方有利的管辖法院 |


⚠️ 乙方友好型管辖:**乙方所在地法院** 或 **乙方所在地仲裁机构**

### 8.3 诉讼费用

- 律师费、诉讼费、保全费由违约方承担?

## 九、合同附件审查

### 9.1 附件完整性

- 合同中提到的所有附件是否齐全?
- 附件编号是否与正文一致?
- 附件是否经双方签字盖章?

### 9.2 附件效力

- 附件与正文同等效力?
- 附件与正文冲突时以何为准?

## 十、签署条款审查

### 10.1 签署形式

- 纸质签署 / 电子签署?
- 电子签署平台是否合法合规?(《电子签名法》要求)

### 10.2 签署主体

- 签字人 + 公章?
- 法定代表人签字 + 公章?
- 授权委托书完备?

### 10.3 生效条件

- 双方签字盖章之日起生效?
- 是否有其他生效条件(如付款、批准等)?
- 多份原件,各执几份?
